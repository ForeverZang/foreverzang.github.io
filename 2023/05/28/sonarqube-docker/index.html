登录SonarQube并配置

  • 访问地址：http://172.16.4.151:9000/，默认账号密码：admin/admin
  • 初次登录需要重置密码：admin/123456

安装Chinese中文插件

  • 选择“Administration” => “Marketplace”，输入Chinese搜索中文汉化包（或者离线下载放到 $PWD/extensions/plugins 目录下）
